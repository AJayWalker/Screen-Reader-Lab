Screen Reader Explained For Mac Users

If you wish to learn about the screen reader on a Mac computer, feel free to use the link below.

Apple's screen reader page

This tutorial covers Apple's screen reader for the Google Chrome web browser while using OS Monterey v12.6.


  • Step 1: To turn on the screen reader press command + fn + F5.
  • Step 2: A tab should appear in the bottom left of your screen along with an voice that starts to read the page you have open.
  • Step 3:
Commanddescription
Control Pressing the 'Control' key will stop the voice over from reading
Tab Selects next link/button on page
Control + Option + Space activates a selected link
Option + Control + A Reads continuously
Option + Control + Right Arrow Skips to the next item on the page
Command + Control + Option + H Skips to next header and reads it aloud

A website that works well with the screen reader is BBC's home page. BBC's page can be traversed with the "Control + Option + H" command which will read the titles of each article that the page currently has on its featured list.


An example of a website that does not work well would be Barnes and Noble's website. The page has too many tabs and as a visitor you are often greeted with a cookies popup which can be difficult for visually impared users to accept or deal with in the case where they do not wish to accept. The multitude of tabs also makes it difficult to navigate because of how cluttered it feels to try and skip through the unnecessary tabs. Switching between using "Command + Control + Option + H" and "Option + Control + Right Arrow" also feels cluttered and difficult to find what is being looked for.
